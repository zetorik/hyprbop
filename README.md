# Installation
## Aur
```bash
yay -S hyprbop
```

# Usage
```bash
hyprbop <configname>
```

# Config
- Config directory: ~/.config/hyprbop/
- Config files: <configname>.json, style.qss
- See example config [power.json](src/hyprbop/config/power.json)
